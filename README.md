🌳 Árvore Binária de Busca em Java (Terminal)
Um projeto prático desenvolvido em Java puro que demonstra a implementação e o funcionamento de uma estrutura de dados do tipo Árvore Binária de Busca (BST). O programa roda de forma interativa diretamente no terminal, permitindo a visualização de diferentes algoritmos de travessia (percurso) após a inserção e remoção de elementos.

⚙️ Funcionalidades
Geração Aleatória: Sorteia e insere automaticamente 20 números únicos (entre 0 e 100) na árvore binária.

Menu Interativo: Um menu de linha de comando para o usuário escolher o método de leitura da árvore.

Algoritmos de Travessia: Implementação dos métodos clássicos de percurso em árvores:

Pré-ordem (Raiz, Esquerda, Direita)

In-ordem (Esquerda, Raiz, Direita - exibe os dados ordenados)

Pós-ordem (Esquerda, Direita, Raiz)

Em nível (Busca em Largura / BFS)

Remoção e Reorganização: Uma função específica que remove 5 nós aleatórios da árvore e refaz todas as impressões, demonstrando como a árvore se reorganiza internamente.

🛠️ Tecnologias Utilizadas
Linguagem: Java (JDK padrão)

Bibliotecas: java.util.* (Scanner, Random, Queue, Set, List)

Estruturas de Dados: Árvore Binária de Busca (BST), Filas (para busca em nível) e Listas.
